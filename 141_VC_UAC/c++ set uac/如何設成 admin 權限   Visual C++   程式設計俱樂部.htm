http://www.cc.ntu.edu.tw/chinese/epaper/0005/20080620_5009.htm

>> 2. compile 程式時，嵌入 "Run as admin" 的 menifest file，或是把
>> menifest file 跟執行檔放在同個目錄下，則執行時，就會要求 user

>> 3. 將需要 admin privileges 才能做的事，放到 NT Service 堨h做。
> 請問放到 NT service 是什麼意思?

M$ 出 UAC 時的其中一個 suggestion，程式需要規劃好那部分需要什麼權限。
因為放在 Service 堛漯F西可在執行時設定執行權限，所以你可以把不需要權
限的東西放在你的 AP 後，需要權限的，透過 IPC 請 Service program 來做。
