在VC埵p何獲得任意兩個日期時間段間的天數？   
  
比如：2001-05-01       2004-07-19   這兩個日期之間間隔多少天，如何通過VC編程實現  

CTime   t1(   2001,   5,   1,   0,   0,   0   );     
CTime   t2(   2004,   7,   19,   0,   0,   0   );     
CTimeSpan   ts   =   t2   -   t1;   
ts.GetDays();
