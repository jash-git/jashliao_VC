Tessellation

會動的密舖圖形

http://blog.controul.com/swf/voronoi1000.swf （有版權爭議，所以就只給個連結。） 可能大家有看過 Voronoi Diagram， 但是沒有想過會動的 Voronoi Diagram 吧？ 於是有人就做了一個這樣的東西， 不斷移動 Voronoi Diagram 每個站點的位置，然後即時算出圖形。 數大便是美，感覺起來很像水紋？ http://nodename.com/wpEmbeds/MonaVoronita/bin/MonaVoronita.swf （有版權爭議，所以就只給個連結。） Voronoi Diagram 看起來像是拼圖， 當然有人就想到說：把一幅圖畫重新用 Voronoi Diagram 拼起來會是什麼樣？ 結果是，有點畢卡索的味道。 如果能弄得更精細一點， 應該會很搶眼的，變成很棒的視覺藝術。 資料來源： [1] http://blog.controul.com/2009/05/speedy-voronoi-diagrams-in-as3flash/ [2] http://nodename.com/blog/2009/05/18/mona-voronita/

馬賽克照片

http://click7.org/image-mosaic-generator/ 用許多張照片拼湊出「蒙娜麗莎的微笑」。 使用的是兩兩配對的概念。 先把「蒙娜麗莎的微笑」切成小格， 再將每一小格與每一張照片計算相似度。 就可以選出最適當的照片了。 這玩意兒有個專有名詞叫作「馬賽克照片」， 網路上已有很多現成軟體，可以玩玩看。

Interactive Art

虛擬動物

原理是以三維座標紀錄動物的身體外圍輪廓， 另外紀錄動物的每個動作會經過的座標， 最後隨機播放動作即可。 要更擬真的話， 可以自行定義動作的相依性， 例如拍拍翅膀後一定要記得搖擺。 如果要讓動作有更細微的變化， 可以設定好有多少機率開始進行下一個動作， 而下一個動作有許多種可能，每一種都設定不同機率， 建立一個 Markov Process 就可以做到了。 要將動物繪於畫面上， 則需要有 Computer Graphics 電腦繪圖的知識。 一般來說，使用 OpenGL / WebGL 內建的函式庫， 就能輕鬆完成3D繪圖。 動物進行轉彎時， 就利用線性代數，將動物輪廓的座標，旋轉一定角度， 就可以使動物面向不同方向。

Body Art

臉部投影

把光線投射到人臉上，產生特效。 再加上臉部辨識、臉部追蹤，那就會更生動一點。 原理？ 請加入影像處理實驗室吧。 :D

臉部運動

究竟是什麼原因， 一個人會想要整治自己的臉呢？ 一個人接上生物電流感應器，連接到電腦。 一個人接上電極，連接到電腦。 這樣就可以用人工方式控制顏面神經， 或者也可以用電腦程式來控制顏面神經。 這個技術將來也可以控制機器人的表情。 最後一定要說： 影片作者係專業人士，好孩子請勿模仿唷！
