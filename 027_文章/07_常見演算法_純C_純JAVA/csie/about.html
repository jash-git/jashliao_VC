About

This site compiles numerous tutorials of basic algorithms. You may need them if you are a computer science student, a software engineer, or an algorithm contest competitor.

This site also collects some online judge problem id, thus you can make practices after reading the tutorials.

These tutorials are wholly written in Traditional Chinese. Although you may not understand the Chinese characters, I hope that you can still find out something useful here, e.g. image and implementation code.

If you have any suggestions, you can contact me at message board. I will reply your message.

工作項目

一、整理國內學校沒有教，但是國外學校有教的演算法。例如高等資料結構、高等演算法、圖論演算法、組合最佳化、計算幾何、字串學等等。

二、整理演算法競賽常見的演算法。以下列出一些知名的演算法競賽：例如專門為資訊科系大學生舉辦的ACM-ICPC，以及知名私人企業舉辦的TopCoder Open: Algorithm、Google Code Jam、Facebook Hacker Cup等等。

三、擷取學術期刊的新演算法，匯整成系統。例如SODA、COCOA、Algorithmica等等。如果讀者發現了觀念十分新穎、應用十分廣泛的演算法，歡迎提供資訊。

四、介紹計算機科學的應用，扭轉一般人對於資訊工程系的迷思──不該只是懂程式語言、懂電腦架構、懂軟體系統而已。

五、取網站內容的精華，出版成冊，作為輔助教材。
