Geographic Information System

GPS衛星導航

就算沒開車的人也應該知道， 現在的汽車幾乎都有安裝衛星導航系統， 要找路？那一點都不困難。 當然啦，找到不塞車的路、車程最短的路， 是滿重要的一件事， 不但省時間、省油錢， 還可以節能減碳愛地球。 身為一個資訊人， 很直覺的就想到演算法課本的「最短路徑」了。 理所當然的， 也就發生了導航到單行道、小巷、死路的問題了。 雖然現在網路上已經能輕易取得道路地圖的資訊， 但是卻沒有道路實際狀況的資訊， 程式進行計算時， 也就沒辦法把路況納入考量範圍內了。 並不是演算法不好，而是資訊不足的問題。 要改善這些問題， 就仰賴科學家與工程師的持續努力吧！
