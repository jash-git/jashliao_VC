Vector Image File Formats: DWG
DWG is a binary file format used by AutoDesk AutoCAD. It can contain 2D or 3D objects and offers compression and a CRC check for internal data.

LEADTOOLS Vector Imaging Pro supports DWG versions 12, 13, 14 and 2000 with the following object types:

3DFACE 
3DSOLID 
ARC 
ATTDEF 
ATTRIB 
CIRCLE 
DIMENSION 
ELLIPSE 
INSERT 
LEADER 
LINE 
LWPOLYLINE 
MTEXT 
POINT 
POLYLINE 
SEQEND 
SOLID 
SPLINE 
TEXT 
TOLERANCE 
TRACE 
VERTEX
