DXF Group Codes
ENTITIES Section
SEQEND

The following group codes apply to seqend entities.

Seqend group codes
Group codes Description

-2

APP: name of entity that began the sequence. This entity marks the end of vertex (vertex type name) for a polyline, or the end of attribute entities (attrib type name) for an insert entity that has attributes (indicated) by 66 group present and nonzero in insert entity).
This code is not saved in a DXF file.
