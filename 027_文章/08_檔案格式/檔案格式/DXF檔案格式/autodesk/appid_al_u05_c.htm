DXF Group Codes
TABLES Section
APPID

The following group codes apply to APPID symbol table entries.

APPID group codes
Group codes Description

100

Subclass marker (AcDbRegAppTableRecord)

2

User-supplied (or application-supplied) application name (for extended data). These table entries maintain a set of names for all registered applications.

70

Standard flag values (See "Common Group Codes for Symbol Table Entries."):
1 = If set, xdata associated with this APPID is not written when SAVEASR12 is performed.

Note   Because there was no change in the ASE data format from R13 to R14, the ASE application name (ACADASER13) did not change.
