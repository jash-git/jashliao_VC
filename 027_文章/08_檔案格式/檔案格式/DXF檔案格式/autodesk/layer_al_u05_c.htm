DXF Group Codes
TABLES Section
LAYER

The following group codes apply to LAYER symbol table entries.

LAYER group codes
Group codes Description

100

Subclass marker (AcDbSymbolTableRecord)

2

Layer name

70

Standard flags. (See "Common Group Codes for Symbol Table Entries.") In addition to the standard flags, the following values apply to layers (bit-coded values)
1 = Layer is frozen, otherwise layer is thawed
2 = Layer is frozen by default in new viewports
4 = Layer is locked

62

Color number (if negative, layer is Off)

6

Linetype name

Xref-dependent layers are output during DXFOUT. For these layers, the associated linetype name in the DXF file is always CONTINUOUS.
