rand
Syntax: 
  #include <stdlib.h>
  int rand( void );

The function rand() returns a pseudorandom integer between zero and RAND_MAX. An example:

   srand( time(NULL) );
   for( i = 0; i < 10; i++ )
     printf( "Random number #%d: %d\n", i, rand() );
