push
Syntax: 
  #include <queue>
  void push( const TYPE& val );

The function push() adds val to the end of the current priority queue.

For example, the following code uses the push() function to add ten integers to the end of a queue:

   queue<int> q;
   for( int i=0; i < 10; i++ )
     q.push(i);
