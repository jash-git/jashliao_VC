getc
Syntax: 
  #include <stdio.h>
  int getc( FILE *stream );

The getc() function returns the next character from stream, or EOF if the end of file is reached. getc() is identical to fgetc(). For example:

   char ch;
   FILE *input = fopen( "stuff", "r" );             

   ch = getc( input );
   while( ch != EOF ) {
     printf( "%c", ch );
     ch = getc( input );
   }
