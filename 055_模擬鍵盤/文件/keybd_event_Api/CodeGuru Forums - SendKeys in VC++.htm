    Re: SendKeys in VC++
    For example - to simulate the Insert key being pressed:

    keybd_event(VK_INSERT, 0, KEYEVENTF_EXTENDEDKEY, 0);
    keybd_event(VK_INSERT, 0, KEYEVENTF_EXTENDEDKEY|KEYEVENTF_KEYUP, 0);
